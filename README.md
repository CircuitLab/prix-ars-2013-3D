#prix-ars-2013-3D

3D projection for Prix Ars 2013 competition.

Requirement
---

###Openframeworks Addon

 - Openframeworks 0073
 [http://www.openframeworks.cc/download/](http://www.openframeworks.cc/download/)

 - ofxOSC

## OSC

 - receive port
   4444
   
 - protocol
 /right/image "filePath" cameraId
 /left/image "filePath" cameraId
